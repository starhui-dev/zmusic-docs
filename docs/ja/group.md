# Discord コミュニティルール {#discord-rules}

[ZMusic Discord サーバー](https://discord.gg/twQgJNufYn)のルールです。参加した時点で、本ルールおよび [Discord コミュニティガイドライン](https://discord.com/guidelines)に同意したものとみなします。

## 質問・ヘルプ {#getting-help}

質問する前に、まず以下を確認してください：

1. [ドキュメント](/ja/guide/introduction)と[よくある質問](/ja/faq)を検索する——そこに既に書かれている質問には回答されない場合があります。
2. **最新バージョン**を使用しているか確認する——その問題はすでに修正されているかもしれません。

問題を報告する際は、以下を添えてください：

| 必要な情報 | 説明 |
| --- | --- |
| バージョン情報 | ZMusic のバージョン、サーバーの種類とバージョン（例：Paper 1.21.1） |
| 完全なログ | エラー発生時の完全なログを [mclo.gs](https://mclo.gs) で共有——チャットに長いログを直接貼らないでください |
| 再現手順 | 何をして、何を期待して、実際に何が起きたか |

::: tip 良い質問の例
❌「音楽が動かない、助けて」

✅「Paper 1.21.1 + ZMusic 4.0.0-dev で `/zmusic play <曲名>` を実行すると xxx と表示されます。完全なログ：https://mclo.gs/xxxxxx」
:::

「いますか？」だけの投稿はせず、問題とすでに試したことをそのまま書いてください。

## ルール {#rules}

1. **敬意を持って接すること**——ハラスメント、ヘイトスピーチ、個人攻撃、脅迫は禁止です。
2. **NSFW、グロテスク、違法なコンテンツの禁止**——テキスト、画像、リンク、ボイスなど形式を問いません。
3. **スパムの禁止**——連投、意味のない長文の貼り付け、絵文字・リアクションの連打など。
4. **無断広告の禁止**——サーバーレンタルや有料サービスなどの宣伝には、事前にモデレーターの許可が必要です。
5. **招待リンクの禁止**——他のサーバーへの招待リンクを無断で共有しないでください。

## 違反への対応 {#enforcement}

モデレーターは状況に応じて判断します。通常の段階は**警告 → タイムアウト → BAN** ですが、悪質な違反は即時 BAN となる場合があります。処置に誤りがあると思う場合は、公開チャンネルで争わず、モデレーターに個別に連絡してください。
