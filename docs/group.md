# 群公告 {#group-announcement}

ZMusic 用户交流群（QQ 群 / 微信群）的共同约定。加群即视为已阅读并同意遵守本页内容。

- 官网：https://zmusic.cc
- 使用文档：https://docs.zmusic.cc

## 提问与求助 {#asking}

### 先自助，再提问 {#self-help}

群里的大部分问题文档都已经写过了。提问之前，请按顺序自查：

1. 搜索[使用文档](/guide/introduction)和[常见问题](/faq)
2. 升级到**最新版本**后再试一次，旧版本的问题可能早已修复
3. 如果你还在用 V2，查 [V2 (LTS) 文档](/v2/)

文档已解答的问题，群里不再重复回复。

### 提问时带上这些信息 {#what-to-provide}

| 需要的信息 | 说明 |
| --- | --- |
| 版本信息 | ZMusic 版本、服务端类型与版本（如 Paper 1.21.1） |
| 完整日志 | 报错时段的完整日志，通过 [mclo.gs](https://mclo.gs) 分享，**不要**在群里直接粘贴大段日志 |
| 复现步骤 | 从输入了什么命令、做了什么操作开始，到出现什么现象，让别人能照着重现 |

::: tip 好的提问长这样
❌「为什么点不了歌？」

✅「Paper 1.21.1 + ZMusic 4.0.0-dev，输入 `/zmusic play 晴天` 后提示 xxx，完整日志：https://mclo.gs/xxxxxx」
:::

提问时直接描述问题和已尝试过的方案，不必先问「在吗」——有人看到自然会回。推荐先阅读[《提问的智慧》](https://starhui.cc/ask.html)。

## 群名片 {#group-card}

- 建议格式：`名字-服务端-版本`，例如 `小明-Paper-1.21.1`
- 名片中禁止添加任何推广信息（联系方式、店铺名、服务器 IP 等）

## 群规 {#rules}

::: danger 红线：违反直接移出，不设警告
1. **反动、政治、色情内容**——包括文字、图片、链接、语音等任何形式。
2. **人肉搜索、威胁恐吓**——针对任何群成员。
3. **引流**——发送其他群号、QQ 频道通行证、群二维码等；分享的图片带有二维码或群号水印且未打码的，视同引流。
:::

::: warning 秩序：警告 → 禁言 → 移出
1. **刷屏、刷图**——短时间内连续发送无意义消息、表情包或图片。
2. **频繁发送色图**——偶尔玩梗不追究，频繁发送按违规处理。
3. **骂街、恶意挑衅**——人身攻击、阴阳怪气、引战等破坏交流氛围的行为。
:::

秩序类违规第一次警告，再犯禁言，屡教不改移出群聊。情节严重或造成恶劣影响的，管理组可直接移出，不受上述梯度限制。本页未列明但明显破坏社区秩序的行为，管理组有权酌情处理。

## 推广 {#promotion}

允许在限制内发布 MC 相关推广（服务器出租、插件定制等）：

1. 每人**每周最多一次**，内容**不超过十行**，不得影响正常聊天。
2. 超出以上限制的，按秩序类违规处理。
3. 因推广产生的任何纠纷与本群无关，请自行甄别、谨慎交易。
